# Phase Arrow

**description**: Weapon (arrow), uncommon This ethereal arrow passes through all matter except the intended target. You ignore all cover, including total cover, when making an attack with this arrow. This arrow can target creatures on the Ethereal Plane as if they were on the Material Plane, and vice versa. Once it hits a target, the arrow is no longer magical.
