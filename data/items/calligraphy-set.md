# Calligraphy Set

**description**: Calligraphy Set Adventuring gear (artisan's tools) These ornate boxes contain items a practiced calligrapher needs to craft books, letters, scrolls, and other written communications. It consists of a small wooden box that contains a variety of brushes, inkstones, and pigments; a small bowl for water; and a small bag of sand for drying. A calligraphy set is required to create wards, charms, and the familiars called shikigami.
