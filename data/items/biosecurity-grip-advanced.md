# Biosecurity Grip (Advanced)

**description**: This device has the same features as the standard version with some additions. The device can register up to 3 authorized users. As a bonus action, a registered user can issue a voice Command to enable or disable the grip’s security feature. An unauthorized creature can bypass or remove the device by succeeding on a DC 17 Intelligence check with a hacker’s kit. On a failed check, the creature takes 4d6 lightning damage.
