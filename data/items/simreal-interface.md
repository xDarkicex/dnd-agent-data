# SIMreal Interface

**description**: SIM Interface Augment (head), uncommon The standard version of this cortical implant is an uncommon augment. As an action, you can activate or deactivate the implant, which functions as either a SIM unit or a SIM AR adapter. You can switch between these two functions as a bonus action. The following SIM interface variants are rare augments and have additional properties. SIMreal. This interface also functions as a SIMreal recorder (see “ Lifestyles in Play ”). A micro-port at the base of your skull is used to house the recorder’s data chip. Virtual Helm Interface. This interface has the added function of a virtual helm and can be linked to one vehicle of your choice (see “ Vehicle Options ”).
