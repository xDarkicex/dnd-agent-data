# Limitless Communicators

**description**: Limitless Communicators Utility, rare These specialized two-way communicators come in pairs. Each one can be used to send and receive messages between any linked limitless communicators in the same dimension as you. You can link multiple communicators during a short rest while keeping them within 10 feet of each other.
