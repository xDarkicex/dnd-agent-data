# Memory Disk

**description**: Wondrous Item, Uncommon This rune-engraved, circular disk has a crystal button at its center and a movable dial at its edge. Record Memory As an Action, you can use the crystal to record everything you can see and hear within 30 feet for up to 10 minutes. The disk can hold 3 memories at a time and, if full, you can use the dial to choose which existing memory is overwritten by a new one. As an Action, you can turn the crystal to project a visible and audible illusion of the selected memory.
