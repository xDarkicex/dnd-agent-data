# Elven Carving Tools

**description**: Elven Carving Tools Wondrous item, rare Engraved with graceful hawks, this finely made set of tools can be used as woodcarver’s tools, carpenter’s tools or tinker’s tools. When used in this way, these tools always grant advantage to any checks made with them. Additionally, when used to help repair a ship, any damage repaired is doubled.
