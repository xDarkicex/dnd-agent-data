# Holo-projector

**description**: Holo-Projector Miscellaneous gear Linking this item to a SIM Unit or Recording Unit allows you to create three-dimensional recordings within a 10-foot square which include both video and audio. The recordings can last up to 3 minutes and are saved onto recording chips which can be played again on any holo-projector.
