# SIM Interface

**description**: SIM Interface Augment (head), uncommon The standard version of this cortical implant is an uncommon augment. As an action, you can activate or deactivate the implant, which functions as either a SIM unit or a SIM AR adapter. You can switch between these two functions as a bonus action.
