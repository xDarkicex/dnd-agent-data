# Infiltrator's Accessory

**description**: Micro-tool Accessory Augment (limb), varies This common augment is a small set of tools implanted into one of your arms. You can extend or retract the tools as a bonus action. While extended, you can use them as if you were holding a micro-toolset. While using the tools, you can’t use that hand for other purposes. Infiltrator’s Accessory. The uncommon version of this augment also includes a set of infiltration tools and a plasma torch.
