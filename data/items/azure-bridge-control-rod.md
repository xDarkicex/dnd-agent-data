# Azure Bridge Control Rod

**description**: The azure bridge control rod was crafted by Master Arlen himself from the same azure stone of which the bridge was built. Once per day as an action, the rod’s wielder can activate the magical defenses of the Azure Bridge to turn the surface of the bridge sticky. Once per day as an action, the effect can be ended. Additionally, once per day, the wielder of the azure bridge control rod can cast the web spell, which conjures a goo-like substance that is inflammable. Once you use this ability of the rod, it can’t be used again until the next dawn.
