# Aldyhn's Game

**description**: Aldyhn’s Game Wondrous item, common Aldyhn created this magical, practice board for the Thinker’s Game and gifted it to her son, Thonir. Though he never became as fond of the game as she, he did use it for a while before abandoning it in favor of more interesting amusements. The board has seven charges. When the item’s owner expends a charge, the board plays an automated game against the owner. The owner, if they are a spellcaster, also regains one expended spell slot of their choice. Once a charge has been spent, a long rest must be finished before another charge is used. Once all seven charges have been used, the owner gains a +1 bonus to any skill checks made while playing the Thinker’s Game (on this board, or otherwise). Once the board has no charges, it regains seven charges the following dawn.
