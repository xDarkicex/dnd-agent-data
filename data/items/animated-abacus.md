# Animated Abacus

**description**: Wondrous item, common If you speak a mathematical equation within 5 feet of this abacus, it calculates the equation and displays the solution. If you are touching the abacus, it calculates only the equations you speak, ignoring all other spoken equations.

**properties**:

- Item Type: Wondrous item
- Item Rarity: common
