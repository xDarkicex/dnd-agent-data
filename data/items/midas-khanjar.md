# Midas Khanjar

**description**: Weapon (Dagger), Common This magic dagger has a blade of pure gold. Golden Touch. As an action, you can utter the dagger's command word and touch it to a nonmagical object no larger than eight cubic inches to turn the object into pure gold. Whenever this property is used, any object previously transformed by the dagger reverts back to its original form. An object's transformation also ends if a Dispel Magic spell or similar magic is used upon it.
