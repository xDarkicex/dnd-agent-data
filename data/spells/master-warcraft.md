# Master Warcraft

**description**: You set forth an arcane stratagem and employ it on all creatures within range, allowing you to move them like pieces on a dragonchess board. All creatures within range make a Wisdom saving throw and willing creatures can automatically fail this save. On a failure, you can command the creature to move up to its speed and take the Attack action or the Cast a Spell action. If you command a creature to use the Attack action, it can only attack with a weapon it is currently wielding. If the creature has the Multiattack ability, you can use this instead of the Attack action. If you use the Cast a Spell action, the creature can only cast a cantrip spell it knows. You choose all targets for attacks and spells cast in this way.
