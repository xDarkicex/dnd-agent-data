# Spell Locket

**description**: Spell Locket Requirements: Level 10 Casting Time: 10 minute ritual Range: touch Duration: unlimited Cost: 1 WP (invested) per level of the spell slot Create an object that will store a spell effect for one use. Any spell caster can cast that spell for the artist. The artist must decide on a spell slot level when creating the object and must be present when a spell is loaded into that slot by a spellcaster (who must actually expend that spell). It is possible to create an object with a certain level spell slot and then at some later date load it with any spell of that or lesser level (but still only one single spell per object). Use of the spell destroys the object (it will crumble). Only the artist can trigger the spell.
