# Restore Pattern

**description**: At your touch, all supernatural markers created by a special attack or power (such as corrupt pattern ) affecting one creature or object are erased. If the object is a forge enhanced weapon, the marker remains, but the effect breaks its owner’s attunement to the object so it can be removed or discarded.
