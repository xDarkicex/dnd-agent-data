# Remote Connection

**description**: You can use a nearby device as if it were right in front of you. Choose any electronic device within range that you can see, is not being worn or carried and that has a user function such as controls or a keypad. When you activate this power, a virtual set of controls appear before you, allowing you to use the device as if you were accessing it physically. You can only use this power on one device at a time. This power has no effect on a device that is currently in use by another user.

**properties**:
