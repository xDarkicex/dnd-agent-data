# Eidetic Memory

**description**: 5th-Level Transmutation (Ritual) Casting Time: 1 action Range: Self Components: V, S, M (a string tied in a knot) Duration: Concentration, up to 1 hour When you cast this spell, you can recall any piece of information you’ve ever read or heard in the past. This ability translates into a +10 bonus on Intelligence checks for the duration of the spell.
