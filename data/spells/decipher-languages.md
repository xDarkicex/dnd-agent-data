# Decipher Languages

**description**: For the duration, you understand the literal meaning of any spoken language that you hear. You also understand any written language that you see, but you must be within 5 feet of the surface upon which the language is written. It takes about 1 minute to read one page of text. This power doesn’t decode secret messages in a text, understand unique symbols, or decrypt programming code that isn’t specifically used for communication purposes only.
